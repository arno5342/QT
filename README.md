# QT
## Cloning the Repository
```
git clone https://github.com/arno5342/QT
```
## Setup
### Task1 Poisson-Gauss approximation and Task 3 Gaussian regression
```
cd Task1_and_Task3
conda env create --file environment.yml
```
Then run Task1_and_Task3.ipynb
### PageRank
```
cd pagerank
conda env create --file environment.yml
```
Then run pagerank.ipynb
